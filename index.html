DIE WURST

Score: 0

Lives: 3

Instructions


To start the game press the space bar, then the pink block will start to move and the game has begun. Using the directional arrow keys you must move the pink block to collide with the red fruit block. This will increase the score and the length of the snake making it harder as the game progresses.

There is also a red poison block that will decrease your lives. You need to make sure you remember which is which. Going into a wall or the snake's own body will end the game and a screen will appear with the option to try again.

The aim of the game is to become as long as possible before dying.
